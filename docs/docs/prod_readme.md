# Clonebnb

[Clonebnb - Heroku](https://clone-bnb.herokuapp.com/)
Clonebnb is a full-stack web application inspired by Airbnb.

**The Technology Stack**
* **Backend:** Ruby on Rails
* **Database:** PostgreSQL
